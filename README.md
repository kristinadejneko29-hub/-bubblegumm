## Описание программы
Expense Tracker - это десктопное приложение для отслеживания личных финансов, позволяющее:
- Добавлять расходы с указанием суммы, категории и даты
- Просматривать список всех расходов в удобной таблице
- Фильтровать расходы по категории и периоду
- Автоматически рассчитывать сумму расходов за выбранный период
- Сохранять данные в JSON-файл
- Удалять ошибочные записи
